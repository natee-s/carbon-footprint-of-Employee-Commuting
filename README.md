# 🍃 Carbon Footprint Tracker (Scope 3 Category 7: Employee Commuting)

## 📌 Overview
Carbon Footprint Tracker is a full-stack web application designed to help organizations track, calculate, and visualize the greenhouse gas emissions generated by employee commuting (Scope 3, Category 7). Built with a clean, minimalist UI, it simplifies the daily check-in process for employees while providing executives with powerful data visualization and reporting tools based on TGO (Thailand Greenhouse Gas Management Organization) standards.

## ✨ Key Features
* **User-Friendly Check-in:** A modern, minimalist interface for employees to log their daily commutes with a time-saving "1-Click Check-in" feature for recurring routes.
* **Real-time Carbon Calculation:** Automatically calculates emissions (kgCO2e) based on vehicle type, fuel type, distance, and carpool/van occupancy.
* **Executive Dashboard:** Interactive charts (trend lines, pie charts) to analyze total emissions, eco-trip percentages, and average carbon per trip. Includes advanced filtering by month, quarter, and transport type.
* **Admin Control Panel:** A dedicated management interface to update Emission Factors (EF), manage company van routes, and review employee travel logs.
* **Export & Reporting:** One-click export of carbon footprint data to **Excel** and **PDF** formats for ESG reporting.

## 💻 Tech Stack
**Frontend:**
* React.js
* Tailwind CSS (Styling & UI)
* Recharts (Data Visualization)
* Lucide React (Icons)
* jsPDF & SheetJS (Data Export)

**Backend:**
* Node.js & Express.js (RESTful API)
* MongoDB & Mongoose (Database)
